Zoteroで文献管理

文献管理にはZoteroを利用しています.

flatpakでのZoteroの導入

以前はDebianパッケージがありましたが, ZoteroのベースとなっているFirrefoxの更新頻度の都合上, 現在はパッケージがありません.

Debian WikiではFlatpakを用いたインストールが紹介されています.

  • Zotero - Debian Wiki
  • FlatPak - Debian Wiki

ここでは, 上記に従い Flatpak 版の Zotero を導入します. 

% sudo apt install flatpak
% flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
% flatpak install flathub org.zotero.Zotero
