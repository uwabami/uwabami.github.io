これは何?

Github で公開しているソフトウェアです． 公開できるモノについては github にどんどん置いています. パッチウェルカム.

RDtool: formatter for RD

RD は Ruby における POD(Plain Old Documentation) のようなものです. RDtool は RD のフォーマッタであり, RD を HTML や roff などに変換できます. 以前は Moonwolf さんがリリースマネジメントをされていましたが, 2011年 5 月に引き継いで, gem 化したりイロイロ弄っています.

Rail: Replace Agent-string Internal Library

しまだみつひろさんが作成された rail を引き取って公開/気がついた時にアップデートしています. お気付きの点がありましたら, ご連絡下さい.

tDiary-mode

Emacsから tDiary を編集するための major mode

TDiary::Style::Org

TDiary::Style::Org は Org-mode の syntax で tDiary.org の記事を書くためのスタイルを提供します．

Icon Symbol Font in Terminal plus

ターミナルでアイコンを表示するためのフォントと, Emacs の domtronn/all-the-icons.el でそのフォントを利用するための elisp.

FSMRMP

自家製プログラミングフォント 'Fantasque Sans Mono' + 'Rounded Mgen+' + twemoji + isfit-plus

Jekyll Org Converter

Jekyll の原稿として Org mode で書かれたファイルを処理できるようにするプラグインです． org ファイルの +OPTIONS を Jekyll の YAML Front Matter として扱える様にしてあります．
