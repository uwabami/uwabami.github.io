RDtool - RD formatter

RDってなに？

Fork me on GitHub

RDはRubyにおけるPOD(Plain Old Documentation)のようなものです。 RDtoolはRDのフォーマッタであり、RDをHTMLなどに変換できます。

Download

stable 版: Gem もしくは tar.gz

  • github tarball
  • http://rubygems.org/gems/rdtool

HEAD: GitHub: uwabami/rdtool

  • HEAD zip
  • HEAD tarball

インストール方法

Gem を使う場合: 

% gem install rdtool

Gem を使わない場合: 

% tar xvzf rdtool-(version).tar.gz
% cd rdtool-(version)
% ruby setup.rb    # or sudo ruby setup.rb
